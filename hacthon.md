## Slide 2 · 我们在解决什么

**电商卖家做短视频带货，被四件事卡死：**

- **贵** — 一条专业带货视频 ¥300-2,000，中小卖家一个月预算就烧完
- **慢** — 人手做一条 2-4 小时，50 个 SKU 排期排不完
- **难** — 跨境卖家要 8 种语言，翻译+配音比视频本身还贵
- **不规模化** — TikTok / 抖音 / Reels 每个平台要不同版本，人工跟不上

> 结果：90% 的卖家还在用静态图 + 九宫格做运营，而算法早已偏向短视频。

## Slide 3 · 我们的方案 · 核心 Pipeline

**四步自动化流程，端到端 < 3 分钟：**

```
爆款分析 ──▶ 产品植入 ──▶ 视频生成 ──▶ 文案/标签/首评自动产出
Gemini 多模态    LLM 改写       主流视频模型       可直接复制发布
```

1. **爆款分析** · Gemini 多模态拆解 TikTok / 抖音 爆款，提炼分镜 / 节奏 / 钩子
2. **产品植入** · 将拆解结构与卖家产品图融合，生成定制化 Sora Prompt
3. **视频生成** · 一键分发到 Sora 2 / VEO 3.1 / Seedance / Hailuo / Grok，多模型并发
4. **成片交付** · 自动产出标题 / 正文 / 5-8 个平台标签 / 首评，零手改

> **单条成本 ~¥1** · **支持 8 种语言** · **失败自动退款**

## Slide 5 · 黑客松期间上新的三大亮点功能

**🆕 GPT-Image-2 商品组图生成**
接入 GPT-Image-2 商品理解模型，一张产品图自动产出 6 种电商风格组图：
生活场景 / 模特展示 / 细节特写 / 平铺摆拍 / 户外场景 / 棚拍风格
内置「目标市场人种约束」— 墨西哥站出拉美模特、东南亚站出东南亚模特，不默认亚洲脸。

**🆕 全球 Stripe + 国内支付宝 双通道支付**
海外用户 Stripe（信用卡 / Apple Pay / Google Pay / ACH / SEPA），
国内用户支付宝，币种自动切换。Webhook 双层幂等，充值即到账。

**🆕 多模型统一调度 + 自动退款**
Sora 2 / VEO 3.1 / Seedance / Hailuo / Grok 可配置化接入，
同步 / 异步 / 阻塞型 API 差异全部封装在 provider 层；
任何一个模型调用失败 → 积分原路自动返还，用户无感知。

## Slide 7 · 技术架构 · 10 分钟能讲清楚的栈

**前端** · Next.js 16 App Router · Turbopack · Zustand · Tailwind
**后端** · Next.js API Routes · Drizzle ORM · Postgres (Supabase)
**AI 层** · Gemini 多模态（视频拆解）· 6 家视频 Provider 适配器 · OpenAI 图像编辑
**基础设施** · Vercel · Cloudflare Worker 上传网关（两 Key 权限模型）· Supabase pg_cron
**支付** · Stripe + 支付宝双通道，Webhook 双层幂等

**核心工程亮点：**
- 任务状态机：`pending → analyzing → generating → polling → done/failed`
- Slot 级交付：批量任务中任何一条失败自动补齐到目标数量
- 分钟级 tick + 300s 预算调度：同步阻塞型 API（Grok ~90s）不拖慢其他任务
- 失败自动退款：积分 / Stripe 事件双重幂等保证
